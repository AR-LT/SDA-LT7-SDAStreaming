# SDAStreaming

Simple music streaming service imitation used for learning purposes.

# Database

### Schema - [schema.sql](db/schema.sql)

To create database table use:

```
mysql -h localhost -u root -p < db/schema.sql
```

### Data - [data.sql](db/data.sql)

To populate database use:

```
mysql -h localhost -u root -p < db/data.sql
```
